Flux in isolation

Microcosm makes it easier to control and modify state. It uses pure, singleton Stores and Actions, keeping all state encapsulated in one place.

Microcosm can be installed with the node package manger, using the following command: 

$ npm install microcosm

Microcosm is a variant of Flux that controls and modifies state in a pure, centralized way.

Stores and action creators are collections of pure functions; they have no data of their own. They tell a Microcosm how it should transform itself from one state to the next.

This design seeks to achieve a reasonable trade off between the simplicity of singletons and the privacy of class instances.
